# 深潜赛事活动后台管理系统功能架构表

## 一级菜单 1：赛事活动管理

| 二级菜单 | 功能描述 |
|---------|---------|
| **活动管理** | **活动列表**（分页列表）：展示活动名称、活动状态（草稿/进行中/已结束）、报名时间、活动时间、报名人数、创建时间；支持按状态、时间筛选；支持关键词搜索<br><br>**新增活动**：基础信息设置（名称、描述、类型）；时间设置（报名开始/结束、活动开始/结束）；KV海报上传；报名设置（费用、人数上限、保险）；报名表单配置（姓名、电话、身份证、紧急联系人等字段勾选）；支付配置；参赛用品设置；活动指引（富文本编辑+自动生成指引二维码）；状态设置<br><br>**编辑活动**：支持修改所有信息；已有报名的活动关键信息修改需二次确认<br><br>**删除活动**：仅草稿可删除；已有报名记录不可删除<br><br>**查看详情**：查看完整信息、报名人员列表（可导出Excel）、签到统计、完赛统计 |
| **报名管理** | **报名记录列表**（分页列表）：展示用户昵称、手机号、活动名称、报名时间、支付状态、签到状态、完赛状态；支持按活动、状态筛选；支持导出Excel<br><br>**报名详情查看**：查看完整报名信息、支付记录、签到时间、完赛时间<br><br>**报名审核**（如需要）：审核通过/驳回；驳回时填写原因 |
| **签到管理** | **二维码生成管理**：为每场活动生成参赛前签到二维码和完赛扫码二维码；支持重新生成；支持下载二维码图片<br><br>**签到记录列表**（分页列表）：展示活动名称、用户信息、签到类型（参赛/完赛）、签到时间；支持按活动、类型筛选；支持导出Excel<br><br>**手动签到/完赛**：运营人员手动标记用户签到/完赛；填写时间、备注 |
| **打卡管理** | **打卡点管理列表**（分页列表）：展示打卡点名称、关联活动、位置信息、创建时间<br><br>**新增打卡点**：打卡点名称、描述；关联活动（可选）；位置信息（地图选点）；生成打卡二维码<br><br>**编辑/删除打卡点**<br><br>**打卡记录查询**：查看用户打卡记录；支持按用户、打卡点、时间筛选 |

## 一级菜单 2：积分管理

| 二级菜单 | 功能描述 |
|---------|---------|
| **积分分类配置** | **积分分类列表**（分页列表）：展示分类名称、分类编码、排序、状态（启用/停用）、创建时间；默认存在：运动积分、文艺积分、赛艇积分、减碳积分<br><br>**新增积分分类**：分类名称、分类编码（唯一）；分类图标上传；分类描述；排序权重；状态设置<br><br>**编辑分类**：支持修改信息；已有积分记录的分类编码不可修改<br><br>**删除分类**：已有积分记录的分类不可删除，仅可停用<br><br>**排序调整**：支持拖拽排序 |
| **积分规则配置** | **积分规则列表**（分页列表）：展示规则名称、积分分类、积分数量、触发条件、状态<br><br>**新增积分规则**：规则名称、描述；关联积分分类；积分数量设置；触发条件配置（扫码领取、社区发布关联减碳标签、其他自定义）；规则有效期；状态（启用/停用）<br><br>**编辑/删除积分规则** |
| **积分发放管理** | **批量发放积分**：选择积分分类；选择用户（导入Excel批量选择或按条件筛选）；设置发放积分数量；填写发放原因/备注；确认发放<br><br>**单用户发放积分**：搜索用户；选择积分分类；输入积分数量（可为负数实现扣减）；填写操作原因；确认发放<br><br>**发放记录列表**（分页列表）：展示操作人、用户信息、积分分类、积分数量、操作原因、操作时间；支持按时间、操作人、分类筛选；支持导出Excel |
| **积分二维码管理** | **积分二维码列表**（分页列表）：展示二维码名称、关联积分分类、积分数量、领取次数限制、已领取次数、状态、创建时间<br><br>**新增积分二维码**：二维码名称、描述；关联积分分类；积分数量；领取限制（每人可领次数、总领取次数上限）；有效期设置；生成二维码并支持下载<br><br>**编辑/删除/停用积分二维码**<br><br>**领取记录查询**：查看每个二维码的领取记录、领取用户、领取时间 |
| **积分日志查询** | **积分变动日志列表**（分页列表）：展示用户信息、积分分类、变动类型（增加/减少）、变动数量、变动前积分、变动后积分、变动原因、触发方式（系统规则/手动发放/扫码领取等）、操作人、操作时间；支持多维度筛选；支持导出Excel<br><br>**用户积分详情查询**：输入用户手机号/昵称查询；展示用户总积分、各分类积分详情；展示该用户所有积分变动记录 |
| **积分排行榜管理** | **排行榜查看**：总积分排行榜（实时）；各分类积分排行榜（实时）；支持按时间段查看（本日/本周/本月/全部）；支持导出Excel |

## 一级菜单 3：商城管理

| 二级菜单 | 功能描述 |
|---------|---------|
| **商品管理** | **商品列表**（分页列表）：展示商品名称、商品分类、库存、销量、上架状态、创建时间；支持按分类、状态筛选；支持关键词搜索<br><br>**新增商品**：基础信息（名称、分类、描述-富文本）；商品图片（轮播图多图上传）；SKU设置（多规格配置，每个SKU独立库存、价格）；价格设置（人民币价格、积分价格-可选择积分分类支持多种积分组合、支付方式：纯积分/纯货币/积分+货币组合）；库存设置（总库存、库存预警值）；兑换标签（限量/新品/热门）；上架设置（立即/定时/暂不上架）；排序权重<br><br>**编辑商品**：支持修改所有信息；已有订单的商品价格修改需二次确认<br><br>**删除商品**：已有订单的商品不可删除，仅可下架<br><br>**批量上下架**：批量操作商品上下架状态 |
| **商品分类管理** | **分类列表**：展示分类名称、排序、状态、商品数量<br><br>**新增分类**：分类名称、分类图标、排序权重<br><br>**编辑/删除分类**：有商品的分类不可删除 |
| **订单管理** | **订单列表**（分页列表）：展示订单号、用户信息、商品信息、支付方式、支付状态、订单状态（待支付/待发货/已发货/已完成/已取消）、下单时间；支持按订单状态、支付方式、时间筛选；支持订单号、用户手机号搜索；支持导出Excel<br><br>**订单详情查看**：查看订单完整信息（商品详情、收货地址、支付信息-积分+货币、物流信息）<br><br>**订单发货**：录入物流公司、物流单号；标记订单为"已发货"状态；系统自动发送发货通知<br><br>**订单取消**：支持手动取消订单；填写取消原因；自动退还积分/货币<br><br>**退款管理**：处理用户退款申请；审核通过后退还积分/货币 |
| **支付配置** | **微信支付配置**：配置微信商户号；配置API密钥；配置支付证书；测试支付连接 |

## 一级菜单 4：内容管理

| 二级菜单 | 功能描述 |
|---------|---------|
| **二维码管理** | **二维码列表**（分页列表）：展示二维码名称、绑定内容类型（图片/富文本/链接/H5页面）、扫码次数、状态、创建时间；支持按内容类型筛选<br><br>**新增内容二维码**：二维码名称、描述；选择内容类型（图片-上传图片、富文本文章-富文本编辑器、赛事指引-关联已配置的赛事指引、外部链接-填写URL、H5页面-上传H5页面入口URL）；有效期设置（可选）；生成二维码并支持下载<br><br>**编辑/删除内容二维码**<br><br>**扫码记录统计**：查看每个二维码的扫码次数、扫码用户、扫码时间 |
| **勋章管理** | **勋章列表**（分页列表）：展示勋章名称、勋章图标、获取条件、获得人数、状态、创建时间<br><br>**新增勋章**：勋章名称、描述；勋章图标上传（多尺寸）；获取条件配置（扫码领取/完成特定活动/达成积分目标/其他自定义）；勋章二维码生成（如选择扫码领取）；状态（启用/停用）<br><br>**编辑/删除勋章**<br><br>**勋章发放记录**：查看勋章发放记录（用户、获得时间、获得方式）；支持手动为用户发放勋章 |
| **社区内容管理** | **POST列表**（分页列表）：展示用户信息、内容预览、发布时间、点赞数、关联标签、审核状态；支持按审核状态、关联标签、时间筛选<br><br>**内容审核**：查看POST完整内容；审核操作（通过/驳回/删除）；驳回/删除时填写原因<br><br>**减碳标签管理**：标签列表（展示所有减碳标签）；新增标签（标签名称、关联积分数量、标签图标）；编辑/删除标签 |
| **轮播图/Banner管理** | **Banner列表**：展示所有Banner（位置-首页/活动页等、图片、跳转链接、排序、状态、生效时间）<br><br>**新增Banner**：选择位置；上传图片；设置跳转链接（活动详情页/商品详情页/外部链接）；排序权重；生效时间段；状态（启用/停用）<br><br>**编辑/删除Banner** |

## 一级菜单 6：客服管理

| 二级菜单 | 功能描述 |
|---------|---------|
| **工单管理** | **工单列表**（分页列表）：展示工单编号、用户信息、问题类型、工单状态（待处理/处理中/已完成/已关闭）、创建时间、处理人；支持按工单状态、问题类型、处理人筛选；支持工单编号、用户手机号搜索<br><br>**工单详情**：查看工单完整信息（用户问题描述、上传的图片/附件）；工单处理记录（时间线展示）<br><br>**工单处理**：分配工单给客服人员；回复用户（支持文字+图片）；更新工单状态；标记工单为已完成/关闭<br><br>**工单统计**：统计工单数量、处理时长、满意度等指标；按问题类型、客服人员统计 |

| **客服电话配置** | **电话号码配置**：设置客服电话号码；设置服务时间；设置语音提示文案 |

## 一级菜单 7：数据统计

| 二级菜单 | 功能描述 |
|---------|---------|
| **运营数据概览** | **核心指标展示**：用户总数、新增用户（本日/本周/本月）；活动总数、进行中活动数、报名总人次；订单总数、交易总额（本日/本周/本月）；积分发放总量、商品兑换次数；社区POST发布数、点赞数<br><br>**数据趋势图表**：用户增长趋势；活动报名趋势；交易额趋势；积分发放趋势 |
| **活动数据统计** | **活动维度统计**：按活动统计（报名人数、签到率、完赛率、用户满意度）；活动对比分析；支持导出报表 |
| **商品销售统计** | **商品销售数据**：商品销量排行；商品兑换方式统计（积分/货币/组合）；库存预警提醒；支持导出报表 |
| **用户行为分析** | **用户画像分析**：用户活跃度分析；用户等级分布；用户积分分布；用户参与活动偏好分析 |

## 一级菜单 8：系统管理

| 二级菜单 | 功能描述 |
|---------|---------|

| **管理员账号管理** | **管理员列表**：展示所有管理员（账号、姓名、手机号、角色、状态、最后登录时间）<br><br>**新增管理员**：账号、密码、姓名、手机号；分配角色；状态（启用/停用）<br><br>**编辑/删除管理员**<br><br>**密码重置**：支持管理员重置密码 |
